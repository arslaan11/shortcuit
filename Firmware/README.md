<h1>Firmware</h1>
<h2>Default Keymap & Layers</h2>
Shortcuit is configured with 3x3 layout plus the rotary encoder.
It utilizes KMK Layers to double the functionality of the keys. Hold the Encoder button to access Layer 1.

<h3>Layer 0(Default - Navigation & Utility)</h3>
<ul>
  <li>Knob: volume up/down/mute(click)</li>
  <li>Key:Copy, Paste, Undo, Desktop toggle, and Arrow Keys</li>
</ul>

<h3>layer 1(App Launcher)</h3>
<ul>
  <li>Keys: instant macro to launch Canva, ChatGPT, GitHub and any other application or any games via OS hotkeys</li>
</ul>
